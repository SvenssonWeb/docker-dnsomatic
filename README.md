Docker DNS-O-Matic
==================

Easily announce your new IP to the world.

Quick Start
-----------

    docker run -d \
        -e USERNAME=${USERNAME} \
        -e PASSWORD=${PASSWORD} \
        -e LAPSE=300 \
        stefaniuk/dnsomatic
